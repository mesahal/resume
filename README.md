# Md Sahal - Portfolio Website

A modern, responsive portfolio website built with React, showcasing professional experience, projects, and achievements.

## Features

- 🎨 Modern UI with dark/light mode toggle
- 📱 Fully responsive design
- ⚡ Smooth animations with Framer Motion
- 🎯 Single-page application with smooth scrolling
- 📧 Contact form with EmailJS integration
- 🎥 Content creation showcase
- 🏆 Achievements and competitive programming stats

## Tech Stack

- **Frontend**: React 18, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React, React Icons
- **Email**: EmailJS
- **Carousel**: Swiper.js
- **Deployment**: Vercel

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm start`
4. Build for production: `npm run build`
